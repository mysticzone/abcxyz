# napkin

A wiki, simple, pure, but colorful.


## run

* install mongodb

* setup virtualenv

```
$ mkdir venv

$ virtualenv venv/

$ source venv/bin/activate
```

* install requirements

```
(venv)$ pip install -r requirements.txt
```

* run server

```
(venv)$ python napkin/server_cherrypy.py
```
or
```
(venv)$ python napkin/run.py
```
